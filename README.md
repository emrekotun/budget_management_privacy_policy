# Privacy Policy

Budget Management is a local-first budgeting app.

## Data Storage

All budget categories, monthly plans, transactions, recurring items, savings goals, debts, and settings are stored on the user’s device using Apple platform storage.

## Network Use

The app does not use a custom backend, analytics service, advertising SDK, tracking SDK, or remote API. It does not phone home.

## CSV Backup And Restore

CSV export creates a user-controlled file through the system share sheet. CSV import reads only the file selected by the user and validates rows before creating transactions.

## Optional iCloud Sync

If iCloud sync is enabled, sync uses Apple’s private CloudKit database for the signed-in iCloud account. No custom server logic is used.

## Data Collection

The app does not collect, sell, or share personal data with the developer or third parties.

## Contact

For privacy questions, contact the app developer at ktnapps@gmail.com.
